# NutriTrack-e - JavaScript Calorie Counter

A lightweight, modern web-based Calorie Counter application designed to allow users to add, monitor, and reset daily food and nutrition logs. Built with semantic structures, clean interactivity, and localized browser data storage.

##  Key Features
* **Dynamic Log Engine**: Instantly append entries to a daily list view and execute quick removals.
* **Live Accumulators**: Computes cumulative calorie totals dynamically, updating the layout in real-time.
* **State Preservation**: Synchronizes all active records with `localStorage` to preserve data across page refreshes.
* **Asynchronous Fetch Simulation**: Connects to a remote mock endpoint to retrieve real-time food item suggestions for quick logging.

##  Project Architecture
* **Structure**: Semantic HTML5 markup layout.
* **Styling**: Modern responsive design using **Tailwind CSS (via CDN)**.
* **Icon Assets**: **Font Awesome v6** icons framework.
* **Logic Engine**: Vanilla JavaScript (ES6+, DOM Manipulation, Async/Await Framework, Event Handling).
